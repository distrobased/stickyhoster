How to Flash Linux ISO

Using Ventoy:

  • Download Ventoy and install to USB
  • Copy ISO files to USB drive
  • Boot from USB and select your distro

Using Rufus:

  • Insert USB, open Rufus
  • Select ISO, choose file system (FAT32/NTFS)
  • Click "Start", then reboot and boot from USB
