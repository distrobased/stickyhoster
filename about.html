👋 About LinuxWall

LinuxWall was born from a simple idea: to make Linux approachable, fun, and accessible for everyone — from first-time users to hardcore sysadmins.

🧑‍💻 Who’s Behind This?

Hey, I'm Sticky! I'm a Linux enthusiast, tinkerer, and open-source advocate. I’ve spent years messing with distros, fixing things I broke, and building tools that actually help people.

🎯 Our Mission

  • Make Linux more beginner-friendly
  • Show off the beauty and power of FOSS
  • Create cool, quirky, and useful tools + games for the Linux ecosystem

🛠️ Tech Stack

This site is built using HTML, CSS, and vanilla JavaScript — staying true to minimalism and performance. Hosted on a Linux server (of course) running Fedora.

📜 The Origin Story

It all started when I couldn’t find a central place for lightweight Linux resources that wasn’t overwhelming or outdated. So, I built one.

🧭 What's Next?

  • More interactive games & puzzles
  • Live distro playground (boot from browser?)
  • Better documentation and community forums
